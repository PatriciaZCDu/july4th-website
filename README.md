# Happy July 4th 🎆

A festive single-page website that displays **"Happy July 4th Everyone"** with red, white, and blue balloons hanging on both sides.

## Run

Just open `index.html` in any browser:

```bash
open index.html
```

## Files

- `index.html` — page markup
- `style.css` — styling, balloons, and sway animation
